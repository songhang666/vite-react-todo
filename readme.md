# react+vite+tailwindcss 实现CRUD-todo应用


## 本地运行

Clone the project

```bash
  git clone https://github.com/songhang666/vite-react-todo.git
```

Go to the project directory

```bash
  cd vite-react-todo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
